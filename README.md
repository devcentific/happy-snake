# Happy Snake

A kid-friendly browser snake game with multiple modes, procedural maps, swipe controls, and accessibility-friendly UI updates.

## Release Notes

### v1.0.0 (2026-02-13)
- Added Kindergarten, Beginner, Intermediate, Advanced, and Insane modes.
- Added mode-based max levels and mode completion flow.
- Added procedural map variation for different level layouts.
- Added swipe controls and optional arrow touch controls.
- Added sound toggle with lightweight effects.
- Added first-launch How to Play overlay.
- Added accessibility live status announcements and canvas fallback text.
- Added safe startup/bootstrap and storage guard handling.
- Added visible Mode/Map HUD labels and build metadata footer.

## Controls
- Keyboard: Arrow keys / WASD
- Pause: Space
- Help overlay: H
- Mobile: Swipe on the board or use touch arrows
